# Medical Suite Edge Application - UI Roadmap

## 1. Core Identity
* **Project Name:** Medical Suite iOS Prototype
* **Vibe:** Clinical, high-contrast, professional, precision-focused.
* **Architecture:** Strict separation of Smart (Riverpod) and Dumb (Stateless/Stateful) widgets.

## 2. Roadmap (Backlog)
- [ ] `device_catalog_sidebar`: Left-hand collapsible inventory showing abstract medical icons.
- [ ] `canvas_workspace`: The interactive 2D drop zone with zoom/pan controls.
- [ ] `annotation_toolbar`: Top bar containing brush size, color pickers, and undo/redo buttons.
- [ ] `export_overlay`: The modal for confirming the PNG capture and saving to iOS Photos.
